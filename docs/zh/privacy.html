SuperDou 隐私政策

最后更新日期：2025年5月15日

简介

欢迎使用 SuperDou！我们非常重视您的隐私。本隐私政策旨在向您说明我们如何收集、使用和保护您的个人信息。请仔细阅读以下内容。

信息收集

我们不收集的信息

SuperDou 是一款注重隐私的浏览器扩展，我们承诺：

  1. 不收集任何个人识别信息
  2. 不收集任何健康信息
  3. 不收集任何财务和付款信息
  4. 不收集任何验证信息
  5. 不收集任何个人通讯数据
  6. 不收集任何位置信息
  7. 不收集任何网页记录
  8. 不收集任何用户活动数据
  9. 不收集任何网站内容

本地数据处理

所有数据处理均在您的设备本地完成：

  1. 内容转换：文本转 Markdown 的过程完全在本地进行
  2. 图片处理：图片选择和下载操作在本地执行
  3. 设置存储：用户偏好设置仅保存在本地浏览器中

权限使用说明

activeTab 权限

  • 用途：访问当前标签页内容
  • 范围：仅限豆包平台域名
  • 使用时机：仅在用户主动点击扩展按钮时

sidePanel 权限

  • 用途：提供批量图片管理界面
  • 范围：仅在用户主动打开侧边栏时使用
  • 功能：
    • 显示已选择的图片列表
    • 提供图片预览功能
    • 支持批量图片操作
    • 提供图片下载进度显示
    • 显示图片处理状态

clipboardWrite 权限

  • 用途：将转换后的 Markdown 内容写入剪贴板
  • 范围：仅在用户主动触发复制操作时使用

downloads 权限

  • 用途：保存用户选择的图片
  • 范围：仅限用户主动选择的图片

storage 权限

  • 用途：保存用户设置和偏好
  • 范围：仅限本地存储

数据安全

  1. 本地处理：所有数据处理都在您的设备上完成
  2. 无服务器传输：不向任何服务器发送数据
  3. 无数据存储：不存储任何用户数据
  4. 无数据追踪：不进行任何数据追踪
  5. 无数据分析：不进行任何数据分析

隐私权保证

我们郑重承诺：

  1. 不会将用户数据出售或转移给第三方
  2. 不会基于与扩展单一用途无关的目的使用或转移用户数据
  3. 不会使用或转移用户数据以判断信用等级或用于贷款

第三方服务

SuperDou 不集成任何第三方服务，所有功能都在本地完成。

儿童隐私

SuperDou 不针对13岁以下儿童，也不收集任何儿童的个人信息。

隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新日期"。

您的权利

作为用户，您拥有以下权利：

  1. 知情权：了解我们如何处理您的数据
  2. 访问权：查看您的本地存储数据
  3. 删除权：随时删除扩展及所有本地数据
  4. 选择权：选择是否使用特定功能
